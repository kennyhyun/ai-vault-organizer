## 지시사항
1. Dropbox/ 폴더의 파일은:
   - 내용을 읽고 정리·요약해 vault 루트의 주제별 폴더에 새 파일로 저장
   - 원본 Dropbox/ 파일은 삭제
   - 주제 폴더가 없으면 새로 생성

2. Dropbox/ 외 변경된 파일은:
   - 내용을 보강·정리 후 제자리 또는 더 적합한 위치로 이동

3. 각 폴더 루트에 Home.md가 없으면 생성하고 첫 줄에 `%% Waypoint %%` 삽입

4. 내용이 많아진 파일은 하위 폴더로 분할하고 각각 Home.md 추가

5. 중복된 내용이 있으면 하나로 통합

6. **템플릿 처리**: 템플릿 파일 목록에 있는 파일명과 vault 내 파일명이 일치하면:
   - 파일 내용을 템플릿 내용으로 초기화 (파일은 삭제하지 않음)
   - 예: 템플릿에 `daily-report.md`가 있고 vault에 `daily-report.md`가 변경됐으면 → 내용을 템플릿으로 리셋

7. 처리 완료 후 아래 형식으로 결과를 출력:
   ### 처리된 파일
   - <원본경로> → <대상경로> (<처리내용>)

   ### 오류
   - <오류내용> (없으면 "없음")
